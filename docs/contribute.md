# How to contribute

### How to clone the repository

```
git clone https://github.com/nutes-uepb/amqp-client-python.git
```

### How to run the lint verification

```
poetry install
```

### How to run the lint verification

```
task lint
```

### How to run the tests

```
task test
```

### How to run the documentation

```
task docs
```
